The Sympy package

Sympy is a Python library for doing symbolic mathematics. We will use it to work with symbolic expressions and use its diff command to take partial derivatives. You can read more about Sympy by following the link above or going straight to its documentation site for tutorials, and manuals, and instructions for how to install it on your own machine.

Q4

In[20]:
f = 1/(sqrt(x**2+y**2))
f
Out[20]:
$\displaystyle \frac{1}{\sqrt{x^{2} + y^{2}}}$
In[21]:
fx = diff(f,x)
fx
Out[21]:
$\displaystyle - \frac{x}{\left(x^{2} + y^{2}\right)^{\frac{3}{2}}}$
In[22]:
fy = diff(f,y)
fy
Out[22]:
$\displaystyle - \frac{y}{\left(x^{2} + y^{2}\right)^{\frac{3}{2}}}$
In[23]:
fx.subs({x:-3,y:4})
Out[23]:
$\displaystyle \frac{3}{125}$

Q5

In[28]:
z = (x+y)/(x-y)
z
Out[28]:
$\displaystyle \frac{x + y}{x - y}$
In[29]:
zx = diff(z,x)
zx
Out[29]:
$\displaystyle \frac{1}{x - y} - \frac{x + y}{\left(x - y\right)^{2}}$
In[30]:
simplify(zx)
Out[30]:
$\displaystyle - \frac{2 y}{\left(x - y\right)^{2}}$
In[31]:
zy = diff(z,y)
zy
Out[31]:
$\displaystyle \frac{1}{x - y} + \frac{x + y}{\left(x - y\right)^{2}}$
In[32]:
simplify(zy)
Out[32]:
$\displaystyle \frac{2 x}{\left(x - y\right)^{2}}$
In[36]:
simplify(x*zx + y*zy)
Out[36]:
$\displaystyle 0$
